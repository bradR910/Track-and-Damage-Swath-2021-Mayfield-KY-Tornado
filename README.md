This project is a geospatial damage analysis of the **December 2021 Western Kentucky EF-4 tornado**. It extracts the tornado's path from a KMZ damage survey file, builds a spatial damage swath by buffering the centerline to the tornado's reported width (~1,600 meters), and then cross-references that swath with OpenStreetMap road network data to identify which roads fell within the damage zone. The analysis culminates in an interactive Folium map centered on Mayfield, Kentucky, with a marked point of interest at the Graves County Courthouse — a landmark heavily damaged in the storm — complete with a popup photo sourced from the National Weather Service.
